📌 Project Title: CRM / ERP Management System
🚀 Overview

This is a CRM/ERP Management System designed to manage business operations like leads, orders, and customers in a structured way. The system helps track sales, manage clients, and streamline workflow efficiently.
✨ Features
👤 Add and manage leads/customers
📊 Track lead status (New, In Progress, Converted)
📦 Manage sales orders
🔍 View structured dashboard of business data
🖥️ Simple and responsive UI
⚡ Fast and lightweight performance
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript (or React if used)
Backend: Node.js / Express (or your stack)
Database: MongoDB / MySQL (if used)
Version Control: Git & GitHub
